# 📊 E-Commerce Sales Data Analysis

## 📌 Project Overview
This project presents a descriptive and exploratory analysis of a 2000-transaction E-Commerce dataset (2023–2025). The objective was to identify sales trends, top-performing regions, category performance, and customer behavior insights.

## 🎯 Business Objectives
- Evaluate overall sales performance
- Identify top-performing categories and regions
- Analyze seasonal sales trends
- Derive strategic business recommendations

## 📁 Dataset Details
- Total Records: 2000
- Regions: 4
- Categories: 5
- Store Types: 3
- Time Span: 2023–2025

## 🛠 Tools Used
- Microsoft Excel
- Power Query
- Pivot Tables
- Data Visualization

## 📊 Key Insights
- North region dominates sales
- Sports & Electronics are top-performing categories
- Online channel contributes 41% of sales
- Strong seasonal peak in Q4
- Female customers show higher premium loyalty tier presence

## 💼 Business Recommendations
- Increase marketing in West region
- Expand inventory for Sports & Electronics
- Strengthen digital payment promotions
- Optimize online channel operations

## 📌 Conclusion
The analysis demonstrates a stable but strategically improvable sales structure with strong growth potential in digital and high-performing categories.
